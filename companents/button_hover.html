BUTTONS HOVER

Formlardaki iletişim kutularında verilen button özelliklerini daha tasarımsal olması için Neo{Labs} tarafından özelleştirilmiştir.

Formlardaki iletişim kutularında yapılan bir diğer değişiklik ise hover renklerini Neo{Labs} tarafında kolayca özellik verilmesi sağlanmıştır. Hover özelliğini hover-{color} yazarak Neo{Labs} oluşturuğu ana renkler yardımı ile tasarımı kolaylaştırmaktır.

Button-hover Yapıları

                                
<div class="d-grid m-auto col-3">
<button class="btn btn-hover">btn-hover-3</button> 
</div>

HTML

                                        
<div class="d-grid m-auto col-3">
<button class="btn btn-hover-2">btn-hover-3</button> 
</div>

</div>

Button Hover ve Renk Seçenekleri

Hoverlı Butonları Default özelliğinin yanı sıra kullanıcılara sunduğumuz yedi farklı renk ile class ekleyerek değiştirebilirsiniz.Örneğin:

  1. hover-black
  2. hover-gray
  3. hover-green
  4. hover-hot
  5. hover-neos-neon
  6. hover-softorange
  7. hover-yellow

Örnek

                    
<button class="btn btn-hover hover-black">btn-hover</button> 
<button class="btn btn-hover hover-gray">btn-hover</button> 
<button class="btn btn-hover hover-green">btn-hover</button> 
<button class="btn btn-hover hover-green">btn-hover</button> 
<button class="btn btn-hover hover-green">btn-hover</button>
