SELECT

HTML'de select etiketi, kullanıcıların belirli bir listeden bir seçenek seçmesine olanak tanır. Select etiketi, option etiketleri ile birlikte kullanılır. Option etiketleri, seçenekleri temsil eder.

Default

Stiller,select öğesinin ilk görünümüyle sınırlıdır ve tarayıcı sınırlamaları nedeniyle option öğelerini değiştiremez.

                                    
                                    
<div class="select-group">
<select class="select-item">
<option selected>Open this select menu</option>
<option value="1">One</option>
<option value="2">Two</option>
<option value="3">Three</option>
</select>
</div>
